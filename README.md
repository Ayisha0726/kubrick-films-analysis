# Kubrick Films Analysis

This project explores the filmography of Stanley Kubrick using basic data analysis and visualization in Python.

## Project Goals

The goal of this project is to analyze trends in Stanley Kubrick's films, including:

- How IMDb ratings changed over time
- Which Kubrick film has the highest rating
- Highlighting the importance of *2001: A Space Odyssey*

## Dataset

The dataset contains information about Kubrick's films including:

- Film title
- Release year
- Runtime
- IMDb rating

## Tools Used

- Python
- pandas
- matplotlib
